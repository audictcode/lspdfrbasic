---
EARLY ACCESS CLUB AUTOMATIC AUTHENTICATION FILE
---
If you want to authenticate with EAC on startup, paste the below XML data into a new file called "earlyaccessclub.xml" and save it.
If you do this correctly, your details will be checked on startup.
DO NOT COPY IN THIS HEADER TO THE XML -- ONLY WHAT IS BELOW THIS SECTION
--

<?xml  version="1.0" encoding="utf-8" ?>
<eac>

  <!-- Make sure you set this to `true` (without backticks) if you want to login on startup! -->
  <authenticateOnStartup>false</authenticateOnStartup>

  <authenticationData>
    <!-- Type your username/beta key into the username field, and your password in the password field. -->
    <!-- Section136Callouts will check with the server for these details at when you go On Duty! -->
    <username></username>
    <password></password>
  </authenticationData>
  
</eac>
